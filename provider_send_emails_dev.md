# provedor de envio de email grátis. Me forneça uma tabela com os melhores inclua o resend

Os melhores provedores gratuitos de envio de email (focando em transacionais para devs, como APIs/SMTP) incluem opções generosas como Brevo e Resend, ideais para automações em n8n ou integrações como Chatwoot.[^1_1][^1_2]

## Comparação de Planos Gratuitos

| Provedor | Emails Gratuitos/Mês | Limite Diário | Destaques para Devs |
| :-- | :-- | :-- | :-- |
| Resend | 3.000 | 100 | API moderna, React Email, webhooks [^1_1][^1_3] |
| Brevo | 9.000 (300/dia) | 300 | API SMTP, analytics, sem CC [^1_2][^1_4] |
| SMTP2GO | 1.000 | 200 | Alta entregabilidade, multi-domínio [^1_5][^1_6] |
| MailerSend | 500 | 100 | API/SMTP, tracking, inbound [^1_7][^1_8] |
| Mailgun | 3.000 (100/dia) | 100 | API robusta, logs, 1 domínio [^1_9] |
| Postmark | 100 | - | Foco transacional rápido [^1_10][^1_11] |

## Recomendações

Para volumes baixos/médios em automações (ex: n8n), priorize Resend pela facilidade dev ou Brevo pela generosidade. Teste APIs via trial; verifique entregabilidade em inbox (SMTP2GO destaca-se). Evite marketing blasts em tiers free para não violar termos.[^1_12][^1_3][^1_6][^1_2]
